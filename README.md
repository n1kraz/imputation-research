# _**Input data and model structures for a modelling study concerning time series missing data**_.

> **Code for the 3rd paper in my PhD** — Repository for reproducible experiments and scripts. 
<br> The purpose of this repository is to provide a transparent, reproducible framework for evaluating and comparing different missing data imputation techniques applied to sensor-acquired water quality datasets.
