# imputation-research
Repo for my machine learning research on data imputation in stormwater dataset

This repository contains code for machine learning algorithms focused on addressing the problem of missing data and data imputation. It includes:

- Implementation of various imputation techniques using machine learning models
- Data preprocessing workflows tailored for datasets with missing values, specifically for the stormwater datasets collected by the author that are also contained in this repo
- Visualization tools for analyzing imputation results and performance accross models

This code is part of my research, which will be presented in the scientific publication discussing the application of machine learning to impute missing data in the context of urban water (stormwater) quality datasets obtained with in-situ sensors. 

To get started, ensure you have the required libraries  installed (requirenments.txt) and follow the usage instructions in this README file.
